# Curl de Backend

Esta carpeta centraliza ejemplos manuales de consumo HTTP por recurso/controlador.

Convencion:
- Un archivo por recurso o controlador.
- Nombre recomendado: `<recurso>.curl.md`.
- Los ejemplos con sesion autenticada deben usar cookies.
- Los ejemplos con body deben incluir datos realistas del dominio.
